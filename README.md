## Program 1: Gladiolus
    Prompt for the number of sections and display
    an ASCII gladiolus flower, with the sections centered
    above each other.
    
### Sample program run:
    Program 1: Gladiolus
    Choose from among the following options:
       1. Display original graphic
       2. Display Gladiolus
       3. Exit the program
    Your choice -> 2
    Number of sections -> 3
    ...
